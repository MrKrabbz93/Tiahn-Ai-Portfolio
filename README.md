## 🏗 Interactive Simulators

Instead of static project descriptions, this portfolio features 4 fully interactive, custom-built UI simulators that demonstrate my capability to build and deploy complex Agentic systems:

1. **Mining Workflow Simulator (`/demos/mining`)**
   - Simulates predictive maintenance and automated IoT routing. 
   - Proves a simulated 35% reduction in manual workflows.
2. **AI Agency Operating System (`/demos/agency`)**
   - Visualizes multi-division autonomous agents handling data streams.
   - Demonstrates 10x throughput in organizational tracking.
3. **No-Code Consultant Platform (`/demos/consultant`)**
   - A mock terminal showcasing concept-to-deployment execution.
   - Represents the ability to deploy enterprise tools in < 2 hours.
4. **Predictive Analytics Platform (`/demos/analytics`)**
   - Shows high-performance dashboards parsing data pipelines.
   - Designed for 40% efficiency gains in large-scale data environments.

## ⚙️ Architecture

*   **Framework:** Next.js 16 (App Router) + React 19
*   **Styling:** Tailwind CSS v4 (Oxide engine + rigid Dark Mode)
*   **Animations:** Framer Motion (Staggered orchestration)
*   **Diagrams:** Client-rendered Mermaid.js 
*   **Deployment:** GitHub Actions -> GitHub Pages (Static Export)
